Lesson 40: Rope Physcis (Java)

Java implemenataion of the original c++ tutorial using Erkin Tunca's base code by Scott Rains

From nehe.gamedev.net: This tutorial was written by Erkin Tunca and is part two of his two part series on physical simulations. Control a rope with keys on your keyboard. Watch as the rope swings and the tail drags across the ground. A very cool effect! The code is based on the latest NeHeGL code and features keyboard interaction and heavy commenting. If you enjoy this tutorial email the author and let him know. He did an amazing job writing both tuts, and released this tut just a few weeks after the first.
